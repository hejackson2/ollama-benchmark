# Ollama benchmark
<br>
<p>This repository provides shell scripts that were vibe coded using various coding harnesses or agents like antigravity, claude, opencode, etc.</p>

<p>The scripts will discover the models already installed on a given ollama server and will run each model sequentially using the same prompt in 'prompt.txt' and save the output from each run to a file unique to the model.  The perfomance data from the '--verbose' flag will be captured and saved to a common CSV file for use in comparing the models against each other.</p>

<p>There is are sample output files "sample__model_comparison__[agy|claude|etc].csv" included in this repository.</p>
